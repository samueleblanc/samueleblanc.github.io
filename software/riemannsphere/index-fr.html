Sphère de Riemann

Ce site permet de voir l'action des transformations de Möbius ou l'action de \(\mathrm{PSp}(4, \mathbb{R})\) sur des cercles, des points, etc. sur la sphère de Riemann. Les méthodes pour calculer ces transformations sont décrites dans les sections correspondantes et en détail dans [Leb23]. Le projet est basé sur les idées développées dans [BK21].

\(\mathcal{Möb}\) | \(\mathrm{PSp}(4, \mathbb{R})\)

Ce site a été créé par Samuel Leblanc, sous la supervision de Jean-Philippe Burelle, dans le cadre du cours Laboratoire de mathématiques expérimentales à l'Université de Sherbrooke. Le code relatif à ce site se trouve sur GitHub. Les visualisations ont été faites avec CindyJS.

[BK21] Jean-Philippe Burelle et Ryan Kirk. Piecewise circular curves and positivity. arXiv:2108.08680, 2021.
[Leb23] Samuel Leblanc. Transformations de cercles orientés tangents sur la sphère de Riemann (PDF). 2023.
